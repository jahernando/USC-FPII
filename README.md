# Particle Physics II — Neutrinos

**Universidade de Santiago de Compostela — Facultade de Física**

Lecture notes on selected topics in Neutrino Physics for the Physics Master programme.

Author: José Ángel Hernando Morata (jose.hernando@usc.es)

## Contents

| Notebook | Topic |
|----------|-------|
| [Introduction](nu_index.ipynb) | The mysteries of the neutrinos |
| [Neutrinos & the SM](nu_sm.ipynb) | History and construction of the Standard Model |
| [Oscillations: Theory](nu_oscillations.ipynb) | Mixing and oscillation theory |
| [Oscillations: Experiments](nu_oscillations_exp.ipynb) | Precision measurements & new generation |
| [Neutrino Mass](nu_mass.ipynb) | Absolute mass scale |
| [Majorana Neutrinos](nu_majorana.ipynb) | Dirac vs Majorana and double-beta decay |

## Usage

The course is published as a [Jupyter Book](https://jupyterbook.org). To build locally:

```bash
jupyter-book build .
```

The online version is available at [GitHub Pages](https://jahernando.github.io/USC-FPII/).
